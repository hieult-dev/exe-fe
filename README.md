# 🚀 FE Project – React App

Frontend application built with **React**, **React-Bootstrap**, and **Tailwind CSS**, focused on clean UI, reusable components, and scalable architecture.

Before run "npm i" to install/update folder node_modules
---

## 📌 Tech Stack

- ⚛️ **React** – UI library
- 🎨 **React-Bootstrap** – UI components
- 💨 **Tailwind CSS** – Utility-first styling
- 🧰 **Vite** / **CRA** – Build tool
- 🌐 **Axios / Fetch** – API communication
- 🧩 **React Router** – Routing
- 🛠 **ESLint + Prettier** – Code quality
